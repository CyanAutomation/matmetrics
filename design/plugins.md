High-Impact Plugins
1. Training Analytics & Insights Plugin

Complement the dashboard's current basic metrics with deeper analysis
Features: Technique frequency trends, effort progression over time, recovery patterns, volume/intensity tracking by technique or category
Natural fit since you already have session data and dashboard structure; this leverages existing Tag Manager and session history

2. Export & Reporting Plugin

Enable users to generate reports (PDF, CSV, JSON) of their training data
Useful for: sharing progress with coaches, analyzing patterns, backing up data outside GitHub
Pairs well with GitHub Sync — users can sync to GitHub and export for external sharing

3. Training Program/Periodization Plugin

Help users plan training blocks with goals (e.g., "focus on grip strength next 4 weeks")
Could track adherence to planned technique focus vs. actual practice
Natural evolution of effort rating — moves from logging what happened to planning what should happen
Supporting Plugins

4. Session Review & Audit Plugin

Allow users to flag sessions for review (incomplete notes, questionable effort ratings)
Auto-detect anomalies (e.g., session marked 5-effort with no techniques logged)
Complements Log Doctor for data quality control

5. Calendar Integration Plugin

Display sessions in a calendar view with heatmaps (intensity, frequency)
Export iCal for Google Calendar/Outlook sync
Helps with training pattern visualization

6. Injury/Recovery Tracker Plugin

Log soreness, injuries, and rest days alongside regular sessions
Track which techniques correlate with discomfort
Essential lifecycle feature for a serious training tool
Lesser Priority (But Useful)

7. Technique Library & Video Linking — Link YouTube tutorials or coaching videos to techniques

8. Sparring Partner Tracking — Log who you trained with and notes about matches/rolls
