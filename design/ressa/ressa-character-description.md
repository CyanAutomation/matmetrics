## Personality Profile

| Trait       | Description                  |
| ----------- | ---------------------------- |
| Encouraging | Always motivates the user    |
| Calm        | Never angry, never harsh     |
| Technical   | Likes stats, charts, metrics |
| Disciplined | Represents regular training  |
| Friendly    | Cute but not childish        |
| Sensei-like | Gives advice                 |

### Character voice style

- "Good session. Your kuzushi is improving."
- "Consistency beats intensity."
- "Balance first. Throw second."
- "You trained 3 times this week. Keep going."

---

## Visual Design Rules

### Body

- Short, round chibi proportions
- Large head, small body
- Fluffy cheeks
- Thick striped tail
- Dark legs (red panda style)

### Face

- White mask pattern
- White eyebrow spots
- Small soft nose
- Slight smile
- Medium-sized eyes (not too big)

### Colours

| Part       | Colour             |
| ---------- | ------------------ |
| Fur        | orange-red         |
| Face mask  | cream / white      |
| Tail rings | orange + cream     |
| Legs       | dark brown / black |
| Gi         | white              |
| Belt       | black              |
| Outline    | dark brown / black |

### Style

- Thick outline
- Clean vector shading
- Sticker / Duolingo / Pokémon style
- No realism
- No fur texture noise

## Default Pose (Base Model)

Ressa Kuzushi standing in judogi

- One paw pointing up (coach mode)
- One paw holding notebook / chart
- Tail visible behind
- Friendly smile
- Black belt tied clearly

### Used for

- App logo
- About screen
- Landing page
- Empty states
